## 🎯 First Time Setup

```bash
# 1. Navigate to project
cd d:\Project Management Tool

# 2. Windows users - run startup script
start.bat

# 3. Or manually - Terminal 1 (Backend)
cd backend
npm install
npm run dev

# 4. Terminal 2 (Frontend)
cd frontend
npm install
npm start

# 5. Open browser
http://localhost:3000

# 6. Create account and start using!
```
