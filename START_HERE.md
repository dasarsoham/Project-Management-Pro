## 💡 CUSTOMIZATION

**Change Colors:**
Edit: `frontend/src/styles/global.css`
```css
--primary-color: #3b82f6;
--secondary-color: #10b981;
```

**Add New Pages:**
1. Create file in `frontend/src/pages/`
2. Add route in `frontend/src/App.js`
3. Add link in navbar

**Extend Backend:**
1. Create controller in `backend/controllers/`
2. Create routes in `backend/routes/`
3. Add to `backend/server.js`
